# Quiz avec Reveal.js

Ce projet contient un mini-site web de quiz avec :

- une page d’accueil avec création de pseudo ;
- 3 exemples de quiz ;
- une question avec une seule réponse ;
- une question à choix multiples ;
- une phrase à compléter ;
- un score final ;
- un classement des utilisateurs.

## Fichiers

- `index.html` : structure du site et des slides Reveal.js
- `style.css` : mise en forme
- `script.js` : logique du quiz, score et classement

## Lancer le site

Ouvrir simplement `index.html` dans le navigateur.

Pour GitHub Pages, envoyer les fichiers dans le dépôt, puis activer Pages sur la branche `main`.
